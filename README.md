# 🦈 ShowShark

ShowShark is a Wireshark dissector focused on **entertainment networking** protocols —  
including sACN, Art-Net, RDMnet, OSC, and IGMP.

It automatically identifies lighting and media devices, displays protocol layers clearly,  
and enhances Wireshark with entertainment-specific context and manufacturer details.

---

## 🔍 Features
- Protocol identification: sACN, Art-Net, RDMnet, OSC, IGMP
- Device manufacturer detection via MAC and ESTA IDs
- Friendly names and entertainment-aware Info columns
- Multicast and querier diagnostics
